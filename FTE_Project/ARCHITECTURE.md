## ğŸ¯ Key Design Decisions

### 1. **Skills-Based Architecture**
- Each skill is independent and reusable
- Skills return structured data for easy integration
- No AI model required (rule-based analysis)

### 2. **Human-in-the-Loop**
- Every action requires explicit approval
- No automatic execution
- Complete transparency

### 3. **File-Based Storage**
- Simple, reliable, no database needed
- Easy to inspect and debug
- Portable across systems

### 4. **Dual-Mode Operation**
- Bronze: Observation only
- Silver: Analysis + approval + execution
- Both can run simultaneously

### 5. **Thread-Based Concurrency**
- WhatsApp watcher runs in background thread
- Folder watcher runs in main thread
- Clean separation of concerns
